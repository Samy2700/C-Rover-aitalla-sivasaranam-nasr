# ROVERRUN PROJECT

## Description

**MARC** est un logiciel de simulation de navigation pour rover développé en langage C dans le cadre d'un projet de l'EFREI (École Française d'Ingénierie). Ce projet permet au rover nommé MARC de naviguer à travers divers terrains en utilisant des algorithmes de recherche de chemin optimisés. Le logiciel lit les données de carte à partir de fichiers, calcule les coûts de déplacement en fonction des types de sol, et optimise le trajet du rover pour atteindre la station de base de manière efficace tout en gérant la consommation d'énergie.

### Fonctionnalités

- **Lecture de Cartes** : Importation de cartes personnalisées définies dans des fichiers `.map`.
- **Gestion des Terrains** : Différents types de sols avec des coûts de déplacement spécifiques.
- **Algorithmes de Pathfinding** : Calcul des chemins optimaux en utilisant une approche de parcours en largeur (BFS).
- **Simulation de Mouvements** : Exécution des mouvements du rover avec mise à jour de la localisation et gestion de l'énergie.
- **Interface Utilisateur** : Sélection de mouvements disponibles et visualisation de la carte avec la position du rover.
